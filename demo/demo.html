CyberG Icons

Downloading and Installation Guide

Install via CDN

Copy and paste the following line into the <head> of your HTML file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/gheorghiciucgeorgia/CyberG-icons@v1.0.1/cyberG.css">

Usage Example

To use an icon, add an <i> tag with the desired class. For example:

<i class="cybg-barcode cybg"></i>

Download Locally

Download from GitHub

After downloading, include the cyberG.css file in your project and link it in your HTML:

<link rel="stylesheet" href="path/to/cyberG.css">

Bellow you can find the icons
